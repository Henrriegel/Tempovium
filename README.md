# Tempovium

Aplicación de escritorio multiplataforma desarrollada en .NET con Avalonia para gestionar contenido multimedia y notas sincronizadas con el tiempo de reproducción.

## ¿Qué hace?

Tempovium permite reproducir contenido (video/audio) y guardar notas asociadas a momentos específicos (timestamps), facilitando el estudio o análisis de contenido.

## Características actuales

- Reproducción de contenido multimedia  
- Notas sincronizadas con el tiempo  
- Importación de archivos a una biblioteca local  
- Detección de archivos duplicados  
- Manejo de base de datos local  

## Tecnologías

- C#
- .NET
- Avalonia UI
- MVVM
- SQLite / LiteDB
